Number Cruncher (working Title)

Number Crunch is a mobile game i’ve been working on recently. I’ve been developing it in Unity. It’s a 2d math puzzle game for Android. Object of the game is to solve multiplication problems using the number squares on screen. There is a time limit so the player need to be quick. Right now I have an early prototype that features the core gameplay. The goal is to publish it on the google play store.
There are still a some features I need to work on such as:

  • leaderboard: So players can compare scores with others.
  • Tutorial: An Interactive way that people can learn how to play the game. Not just a page of text.

This game has a lot of opportunity for expansion. I can easily see different game modes added.

Past Work

PartDropper - Unity - Q2 2015

Part Dropper is a 2D action puzzle game. The goal of the game is to catch parts that are shot into the air and then drop them into their construction area. Once all the parts are place Budd-E will go over and start constructing a vehicle. If you miss the part will fall on the ground and break. Budd-E will then go over and sweep up the part then go to the recycle bin and dump it. The game ends once the timer runs out afterward your score will be added up based on how you place the parts and how many cars you’ve completed. You can then submit your score to the games leaderboard. During development I would with two artist(illustrator and animator) to create the assets for the game.
