Current Work

Number Cruncher (working Title) - Unity

Number Cruncher is a mobile game i’ve been working on recently. I’ve been developing it in Unity. It’s a 2d math puzzle game for Android. Object of the game is to solve multiplication problems using the number squares on screen. There is a time limit so the player need to be quick. Right now I have an early prototype that features the core gameplay. The goal is to publish it on the google play store.
There are still a some features I need to work on such as:

  • leaderboard: So players can compare scores with others.
  • Tutorial: An Interactive way that people can learn how to play the game. Not just a page of text.

This game has a lot of opportunity for expansion. I can easily see different game modes added.

Past Work

PartDropper - Unity - Q2 2015

Part Dropper is a 2D action puzzle game. The goal of the game is to catch parts that are shot into the air and then drop them into their construction area. Once all the parts are place Budd-E will go over and start constructing a vehicle. If you miss the part will fall on the ground and break. Budd-E will then go over and sweep up the part then go to the recycle bin and dump it. The game ends once the timer runs out afterward your score will be added up based on how you place the parts and how many cars you’ve completed. You can then submit your score to the games leaderboard. During development I would with two artist(illustrator and animator) to create the assets for the game.

The Choice - Unity - Q4 2013

This was an Action Role Playing Game I made as final project for a programming class I took while going to The Academy of Art. In the game you play a magical being capable of transforming into different classes(assassin, warrior, and archer). Each class has a special ability that you can use when you switch to them. Assassin can backstab, warrior is a tank, and the archer can shoot enemies from a distance. The game features 5 locations: cave(assassins hideout), forest(archers hang out), castle(where the warriors guard), main town(where you accept quests and shop), and an over world. Throughout the game you will also be faced with making decision based on the quests you accept. These choices will give you a different ending based on what you chose.
