Number Cruncher (working Title)

Number Crunch is a mobile game i’ve been working on recently. I’ve been developing it in Unity. It’s a 2d math puzzle game for Android. Object of the game is to solve multiplication problems using the number squares on screen. There is a time limit so the player need to be quick. Right now I have an early prototype that features the core gameplay. The goal is to publish it on the google play store. There are still a some features I need to work such as: - leaderboard: So players can compare scores with others. - Tutorial: An Interactive way that people can learn how to play the game. Not just a page of text. This game has a lot of opportunity for expansion. I can easily see different game modes added.
